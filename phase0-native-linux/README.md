# Phase 0 Native Linux Results

This branch stores published result artifacts from the hosted native-Linux Phase
0 workflow.

Expected contents:
- `latest/run_summary.json`
- `latest/phase0_packet.json` when packet generation succeeds
- `latest/error_summary.json` when the run fails before packet generation

These files exist only to make the Phase 0 gate review fetchable without GitHub
UI authentication.
